# 숫자 야구 구현할 기능 목록을 정리

### MVC 패턴에 따라 model, view, controller 별로 구현해야하는 기능을 나누어 정리 

### 사용자에게 입력 받아야 하는 경우(InputView)
- [ ] 3자리 수를 입력(숫자를 입력해주세요)
    - [ ] 예외처리(IllegalArgumentException 발생 후 종료)
        - [ ] 숫자가 아닐경우
        - [ ] 3자리수가 아닐 경우
        - [ ] 각 자리 숫자가 중복일 경우
- [ ] 게임 재시작 또는 종료 선택
    - [ ] 예외처리(IllegalArgumentException 발생 후 종료)
        - [ ] 1,2가 아닌 다른 숫자 입력
        - [ ] 숫자가 아닌 경우
